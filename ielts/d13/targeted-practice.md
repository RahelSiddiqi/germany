## üìñ Reading Practice Sets

### True/False/Not Given Practice

**Strategy Reminder:**
| Answer | Meaning |
|--------|---------|
| TRUE | Text confirms statement |
| FALSE | Text contradicts statement |
| NOT GIVEN | No information in text |

**Practice Technique:**

1. Underline keywords in question
2. Find location in text
3. Compare carefully
4. If no info ‚Üí NOT GIVEN

**10 Practice Statements:**
(Use Cambridge 15 Test 3 passage)

1. Statement: **\_**
    - Keywords to find: **\_**
    - Text location: para \_\_\_
    - Answer: **\_**

2-10. (Continue pattern)

---

### Matching Headings Practice

**Strategy Reminder:**

1. Read headings first
2. Identify paragraph main idea
3. Don't match on single words
4. Cross out used headings

**Common Traps:**

-   Word matching (same word ‚â† correct answer)
-   Detail vs main idea
-   First sentence isn't always the topic

**Practice:**
Choose 2-3 passages with matching headings questions.

---

### Sentence Completion Practice

**Strategy Reminder:**

1. Read sentence first
2. Predict answer type (noun? verb?)
3. Find location in text
4. Check grammar matches
5. Check word limit

**Practice Focus:**

-   Grammar accuracy
-   Spelling
-   Word limits

## ‚úçÔ∏è Writing Practice Sets

### Task 1 Intensive Practice

**If Overview was weak:**

Write overviews for these scenarios:

1. Line graph showing population change
   Overview: **\_**

2. Bar chart comparing cities
   Overview: **\_**

3. Pie charts showing changes
   Overview: **\_**

**Good Overview Formula:**
"Overall, [main trend 1], while [main trend 2]."

---

**If Data Description was weak:**

Practice describing:

-   Increases: rose, increased, grew, climbed, soared
-   Decreases: fell, declined, dropped, decreased, plummeted
-   Stability: remained stable, stayed constant, plateaued

**Practice Exercise:**
Write 5 sentences describing data changes with variety.

---

### Task 2 Intensive Practice

**If Introduction was weak:**

**Formula:**

1. Paraphrase question (1 sentence)
2. State position/thesis (1 sentence)
3. Outline main points (optional, 1 sentence)

**Practice:**
Rewrite your Mock Test introduction 3 times.

---

**If Body Paragraphs were weak:**

**PEEL Structure:**

-   **P**oint: Topic sentence
-   **E**xplanation: Develop the point
-   **E**xample: Specific evidence
-   **L**ink: Connect to next point

**Practice:**
Write 3 body paragraphs using PEEL structure.

---

**If Conclusion was weak:**

**Formula:**

1. Restate position
2. Summarize main points
3. NO new ideas

**Practice:**
Write 5 conclusions for different essays.

---

### Part 2 Practice

**Time Management:**

-   1 min planning ‚Üí brief notes only
-   2 min speaking ‚Üí cover ALL points

**Practice Cue Cards:**

**Card 1:**

> Describe a skill you learned as a child.
>
> -   What skill it was
> -   Who taught you
> -   How you learned it
> -   Why it was important

**Card 2:**

> Describe a place you would like to visit.
>
> -   Where it is
> -   What you know about it
> -   Why you want to go
> -   When you plan to visit

**Card 3:**

> Describe a time you received good advice.
>
> -   What the advice was
> -   Who gave it
> -   When you received it
> -   How it helped you

---

### Part 3 Practice

**Answer Structure:**

1. State opinion
2. Explain why
3. Give example
4. Consider other side

**Practice Questions:**

**On Education:**

1. How has education changed in recent years?
2. Do you think online learning is effective?
3. Should education focus more on practical skills?

**On Technology:**

1. How has technology changed communication?
2. Are people too dependent on technology?
3. What are the disadvantages of technology?

**On Environment:**

1. What can individuals do to protect the environment?
2. Should companies be responsible for environmental protection?
3. How can governments encourage eco-friendly behavior?
