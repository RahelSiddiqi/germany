## üìã Question Type 1: MATCHING HEADINGS

### What It Tests

-   Understanding main idea of each paragraph
-   Distinguishing between main point and details

### Strategy

**Step 1: Read ALL headings first** (1-2 min)

-   Underline key words
-   Note themes and topics
-   Look for contrasts/opposites

**Step 2: Skim each paragraph**

-   Read first 2 sentences carefully
-   Read last sentence
-   Identify the MAIN IDEA (not details)

**Step 3: Match and eliminate**

-   Choose best match
-   Cross off used headings
-   Don't second-guess too much

### Common Traps

| Trap                               | How to Avoid                           |
| ---------------------------------- | -------------------------------------- |
| Same word in heading and paragraph | Same word ‚â† correct! Check meaning     |
| Detail vs. main idea               | Ask: "What is this paragraph ABOUT?"   |
| Two headings seem to fit           | One is usually a detail, not main idea |

### Heading Patterns

| If heading mentions... | Paragraph probably discusses... |
| ---------------------- | ------------------------------- |
| Origins/History        | How something began             |
| Causes/Reasons         | Why something happens           |
| Effects/Impact         | Results/consequences            |
| Solution/Response      | How to fix a problem            |
| Challenges/Problems    | Difficulties faced              |
| Benefits/Advantages    | Positive aspects                |
| Drawbacks/Limitations  | Negative aspects                |
| Future/Outlook         | Predictions/prospects           |

### Practice: Match Headings

**Headings:**
i. The environmental cost of progress
ii. A new approach to conservation
iii. Historical context of the issue
iv. Economic factors driving change
v. Limitations of current solutions
vi. Government response

**Paragraph A:**

> The industrial revolution of the 18th century marked a turning point in human history. Factories emerged across Europe, powered first by water and then by coal. This period saw unprecedented growth in manufacturing, but it also initiated patterns of resource extraction and environmental damage that continue to this day.

**Best heading for Paragraph A:** **\_** (Answer: iii)

## üìã Question Type 2: TRUE/FALSE/NOT GIVEN

### Key Definitions

| Answer        | Definition                 | Example                                                        |
| ------------- | -------------------------- | -------------------------------------------------------------- |
| **TRUE**      | Statement agrees with text | Text: "Water is essential" ‚Üí Statement: "Water is necessary" ‚úì |
| **FALSE**     | Statement contradicts text | Text: "Water is essential" ‚Üí Statement: "Water is optional" ‚úó  |
| **NOT GIVEN** | Information not in text    | Text: "Water is essential" ‚Üí Statement: "Water is cheap" ?     |

### Step-by-Step Strategy

1. **Read statement** carefully - underline keywords
2. **Locate** relevant section in text
3. **Compare** statement with text EXACTLY
4. **Decide:**
    - Same meaning? ‚Üí TRUE
    - Opposite meaning? ‚Üí FALSE
    - Topic mentioned but not this specific claim? ‚Üí NOT GIVEN

### FALSE vs. NOT GIVEN

| FALSE                  | NOT GIVEN                                  |
| ---------------------- | ------------------------------------------ |
| Text says the opposite | Text doesn't mention this at all           |
| Direct contradiction   | Related topic, but not this specific point |
| You can prove it wrong | You cannot prove it right OR wrong         |

### Practice Examples

**Text:**

> Coffee originated in Ethiopia and was first cultivated in Yemen in the 15th century. By the 17th century, coffee houses had become popular meeting places in major European cities. Today, Brazil is the world's largest coffee producer, accounting for approximately 35% of global production.

**Statements:**

1. Coffee was first grown in Ethiopia.

    - **Answer:** FALSE (cultivated in Yemen, not Ethiopia)

2. Coffee houses existed in Europe before the 18th century.

    - **Answer:** TRUE (17th century = before 18th)

3. Brazil produces more coffee than all other countries combined.

    - **Answer:** NOT GIVEN (35% is not "more than all others combined")

4. Coffee consumption has increased over the centuries.
    - **Answer:** NOT GIVEN (popularity mentioned, not consumption amounts)

## üìã Question Type 3: YES/NO/NOT GIVEN

### Difference from T/F/NG

| T/F/NG                | Y/N/NG                          |
| --------------------- | ------------------------------- |
| Facts and information | Writer's opinions and views     |
| "The text says..."    | "The writer believes/claims..." |

### Key Phrases for Writer's Views

| Phrase                    | Indicates Opinion            |
| ------------------------- | ---------------------------- |
| "The author argues..."    | Strong view                  |
| "The writer suggests..."  | Implied view                 |
| "It is claimed that..."   | Reported view                |
| "Critics/Some believe..." | Others' views (not author's) |

### Practice

**Text:**

> The government's new education policy has been described as ambitious by many experts. However, I believe that without adequate funding, these reforms will fail to achieve their stated goals. While the intentions are admirable, the practical implementation leaves much to be desired.

**Statements:**

1. The writer supports the government's education policy.

    - **Answer:** NO ("will fail", "leaves much to be desired")

2. The writer thinks the policy goals are realistic.

    - **Answer:** NO ("without adequate funding... will fail")

3. The writer has experience in education.
    - **Answer:** NOT GIVEN (not mentioned)

## üìã Question Type 4: SENTENCE COMPLETION

### Strategy

1. **Read the incomplete sentence** - note grammar needed
2. **Identify keywords** - find location in text
3. **Read around** - answer is usually nearby
4. **Check word limit** - don't exceed
5. **Check grammar** - must fit the sentence

### Grammar Clues

| Sentence ends with... | Answer is probably...                         |
| --------------------- | --------------------------------------------- |
| a/an                  | Singular noun (starting with vowel/consonant) |
| the                   | Specific noun                                 |
| to                    | Verb (infinitive)                             |
| by                    | Agent/method                                  |
| of                    | Noun                                          |
| Verb + -ing           | Present participle                            |

### Practice

**Text:**

> The new recycling program requires residents to separate their waste into three categories: paper, plastic, and organic matter. Collection takes place every Tuesday morning, and special containers are provided free of charge to all participating households.

**Complete the sentences:**

1. Residents must divide waste into **\_** types.

    - Answer: **three**

2. Waste is collected on **\_** mornings.

    - Answer: **Tuesday**

3. Containers are given to households at no **\_**.
    - Answer: **cost** (or "charge")

## üìã Question Type 5: MULTIPLE CHOICE (MCQ)

### Types of MCQ

1. **Choose ONE answer** (A, B, C, or D)
2. **Choose TWO answers** from list
3. **Choose the correct ending** for a sentence

### Strategy

1. **Read question stem** carefully
2. **Underline keywords** in all options
3. **Locate** relevant section in text
4. **Eliminate** wrong options
5. **Check** remaining option matches text

### Common Traps

| Trap              | Example                                                 |
| ----------------- | ------------------------------------------------------- |
| Partially correct | Option mentions something from text but adds wrong info |
| Too extreme       | "always", "never", "all" when text says "often", "some" |
| Opposite meaning  | Negatives and contradictions                            |
| Different topic   | Option from wrong part of passage                       |

### Practice

**Text:**

> Solar panels have become increasingly affordable in recent years, with costs falling by approximately 70% since 2010. While they remain more expensive than traditional energy sources in many regions, government subsidies have made them competitive in several European countries. Experts predict that solar energy will become the cheapest energy source globally by 2030.

**Question: According to the passage, solar energy**
A) is currently the cheapest energy option worldwide
B) has become less expensive since 2010
C) is only affordable in European countries
D) will never be competitive with fossil fuels

**Answer:** B

-   A is wrong (prediction for 2030, not current)
-   C is wrong (several European countries, not only)
-   D is wrong (text predicts it will be cheapest)
