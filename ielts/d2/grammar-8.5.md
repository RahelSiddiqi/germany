## üìñ Essential Grammar Structures

### 1. Complex Sentences

#### Relative Clauses (Defining & Non-Defining)

| Type         | Structure   | Example                                              |
| ------------ | ----------- | ---------------------------------------------------- |
| Defining     | No commas   | The students **who study regularly** perform better. |
| Non-defining | With commas | My brother, **who lives in London**, is a doctor.    |
| Reduced      | Participle  | The report **published yesterday** shows growth.     |

**Practice Sentences:**

-   People **who exercise daily** tend to live longer.
-   Climate change, **which affects all nations**, requires global cooperation.
-   The policy **introduced last year** has reduced emissions.

---

### 2. Conditional Sentences

| Type   | Structure                          | Example                                                       | Use                        |
| ------ | ---------------------------------- | ------------------------------------------------------------- | -------------------------- |
| Zero   | If + present, present              | If water reaches 100¬∞C, it boils.                             | Facts/General truths       |
| First  | If + present, will + base          | If governments invest more, education will improve.           | Real/Possible future       |
| Second | If + past, would + base            | If I were the president, I would prioritize healthcare.       | Hypothetical present       |
| Third  | If + past perfect, would have + PP | If they had acted sooner, the crisis would have been avoided. | Hypothetical past          |
| Mixed  | Various combinations               | If we had invested earlier, we would be wealthy now.          | Past cause, present result |

**Band 8.5+ Variations:**

-   **Were + subject (Inversion):** _Were the government to increase funding, results would improve._
-   **Should + subject:** _Should you require assistance, please contact us._
-   **Had + subject:** _Had they known the consequences, they would have acted differently._

---

### 3. Passive Voice

| Tense              | Active                             | Passive                                |
| ------------------ | ---------------------------------- | -------------------------------------- |
| Present Simple     | People speak English globally.     | English **is spoken** globally.        |
| Present Continuous | They are building new schools.     | New schools **are being built**.       |
| Present Perfect    | Scientists have discovered a cure. | A cure **has been discovered**.        |
| Past Simple        | The storm destroyed homes.         | Homes **were destroyed** by the storm. |
| Future             | They will implement changes.       | Changes **will be implemented**.       |
| Modal              | We must address this issue.        | This issue **must be addressed**.      |

**Advanced Passive Structures:**

-   **Get passive:** _The project got delayed due to funding issues._
-   **Have something done:** _The government had the roads repaired._
-   **Passive reporting verbs:** _It is widely believed that..._ / _Education is considered to be..._

---

### 4. Participle Clauses

| Type               | Structure   | Example                                                         |
| ------------------ | ----------- | --------------------------------------------------------------- |
| Present Participle | -ing        | **Having completed** the research, they published the findings. |
| Past Participle    | -ed/V3      | **Faced with** rising costs, companies reduced staff.           |
| Perfect Participle | Having + V3 | **Having been warned**, they took precautions.                  |

**Usage in IELTS Writing:**

-   **Giving reasons:** _Being a developing nation, India faces infrastructure challenges._
-   **Showing result:** _The economy grew rapidly, resulting in higher employment._
-   **Adding information:** _Written in 1984, the novel remains relevant today._

---

### 5. Cleft Sentences (Emphasis)

| Type           | Structure                    | Example                                        |
| -------------- | ---------------------------- | ---------------------------------------------- |
| It-cleft       | It + be + focus + that/who   | **It is education that** transforms societies. |
| What-cleft     | What + clause + be + focus   | **What we need is** immediate action.          |
| All-cleft      | All + clause + be + focus    | **All they want is** equal opportunities.      |
| The thing that | The thing that + clause + be | **The thing that concerns me is** pollution.   |

**Band 8.5+ Examples:**

-   _It was not until the 20th century **that** women gained voting rights._
-   _What makes this issue complex **is** the lack of consensus._
-   _The reason **why** governments hesitate **is** the economic cost._

---

### 6. Inversion for Emphasis

| Trigger             | Structure                  | Example                                                         |
| ------------------- | -------------------------- | --------------------------------------------------------------- |
| Negative adverbs    | Neg + aux + subject + verb | **Never before have** we faced such challenges.                 |
| Only + time         | Only + time + aux + S + V  | **Only when** we cooperate **can** we succeed.                  |
| Not only...but also | Not only + aux + S + V     | **Not only does** it save money, **but it also** reduces waste. |
| Little              | Little + aux + S + V       | **Little did** they realize the consequences.                   |
| Seldom/Rarely       | Seldom + aux + S + V       | **Seldom do** we see such dedication.                           |
| No sooner...than    | No sooner + had + S + V3   | **No sooner had** they arrived **than** it started raining.     |

---

### 7. Noun Clauses

| Function   | Structure                | Example                                              |
| ---------- | ------------------------ | ---------------------------------------------------- |
| Subject    | That/What/Whether clause | **What governments decide** affects everyone.        |
| Object     | That/Wh- clause          | Research shows **that pollution causes illness**.    |
| Complement | That clause              | The problem is **that resources are limited**.       |
| Appositive | That clause after noun   | The fact **that climate is changing** is undeniable. |

**Useful Patterns:**

-   _It is evident/clear/obvious **that**..._
-   _There is no denying **that**..._
-   _The question of **whether**... remains debated._

---

### 8. Comparatives & Superlatives (Advanced)

| Structure                            | Example                                                      |
| ------------------------------------ | ------------------------------------------------------------ |
| The + comparative, the + comparative | **The more** people exercise, **the healthier** they become. |
| By far the + superlative             | This is **by far the most** significant factor.              |
| Comparative + and + comparative      | The situation is becoming **more and more** serious.         |
| No + comparative + than              | This method is **no better than** the previous one.          |
| As + adj + as                        | The results were **as good as** expected.                    |
| Not nearly as...as                   | This policy is **not nearly as effective as** claimed.       |

---

### 9. Modal Verbs for Academic Writing

| Modal     | Function                    | Example                                        |
| --------- | --------------------------- | ---------------------------------------------- |
| May/Might | Possibility                 | This **may** lead to improvements.             |
| Could     | Tentative suggestion        | Governments **could** implement stricter laws. |
| Should    | Recommendation              | More research **should** be conducted.         |
| Must      | Strong obligation/Deduction | Action **must** be taken immediately.          |
| Would     | Hypothetical                | This **would** require significant investment. |
| Can       | Ability/General possibility | Technology **can** solve many problems.        |

**Hedging (Essential for Band 8.5+):**

-   _It **could be argued** that..._
-   _This **would appear to** suggest..._
-   _There **may well be** other factors..._
-   _It **is likely/unlikely** that..._

---

### 10. Conjunctions & Connectors

#### Coordinating (FANBOYS)

| Conjunction | Use               | Example                                      |
| ----------- | ----------------- | -------------------------------------------- |
| For         | Reason            | He stayed home, **for** he was ill.          |
| And         | Addition          | Education **and** healthcare are priorities. |
| Nor         | Negative addition | He didn't call, **nor** did he write.        |
| But         | Contrast          | It's expensive **but** effective.            |
| Or          | Alternative       | Reduce emissions **or** face consequences.   |
| Yet         | Contrast          | Simple **yet** effective solutions exist.    |
| So          | Result            | Costs rose, **so** prices increased.         |

#### Subordinating

| Function  | Conjunctions                                  | Example                                                  |
| --------- | --------------------------------------------- | -------------------------------------------------------- |
| Time      | when, while, as, before, after, until, since  | **While** technology advances, problems persist.         |
| Reason    | because, since, as, given that                | **Since** resources are limited, priorities must be set. |
| Contrast  | although, though, even though, whereas, while | **Although** expensive, the investment is worthwhile.    |
| Condition | if, unless, provided that, as long as         | **Unless** action is taken, problems will worsen.        |
| Purpose   | so that, in order that                        | Laws exist **so that** society functions.                |
| Result    | so...that, such...that                        | The impact was **so significant that** reforms followed. |

## üìù Writing Task 2 Grammar Templates

### Introduction Patterns

```
It is often argued that [topic]. While some believe [view 1],
others contend that [view 2]. This essay will discuss both
perspectives before presenting my own view.
```

### Body Paragraph Structures

```
One significant advantage/disadvantage is that [point].
This is because [reason]. For instance, [example].
As a result, [consequence].
```

### Concession + Counter-argument

```
Admittedly, it could be argued that [opposing view].
However, this argument fails to consider [your point].
In reality, [explanation].
```

### Conclusion Patterns

```
In conclusion, while [acknowledge opposing view],
I firmly believe that [your position] because [reason].
Governments/Individuals should [recommendation].
```

## ‚ö†Ô∏è Common Grammar Errors to Avoid

### 1. Subject-Verb Agreement

| ‚ùå Wrong                                            | ‚úÖ Correct                                               |
| --------------------------------------------------- | -------------------------------------------------------- |
| The number of students have increased.              | The number of students **has** increased.                |
| Everyone have their own opinion.                    | Everyone **has** their own opinion.                      |
| Neither the government nor citizens is responsible. | Neither the government nor citizens **are** responsible. |

### 2. Article Usage

| ‚ùå Wrong                              | ‚úÖ Correct                          |
| ------------------------------------- | ----------------------------------- |
| Education is the key to success.      | Education is **a** key to success.  |
| The technology has transformed lives. | Technology has transformed lives.   |
| I went to university in UK.           | I went to university in **the** UK. |

### 3. Tense Consistency

| ‚ùå Wrong                         | ‚úÖ Correct                                   |
| -------------------------------- | -------------------------------------------- |
| He said he will come.            | He said he **would** come.                   |
| If I would have known, I helped. | If I **had** known, I **would have** helped. |

### 4. Word Order

| ‚ùå Wrong                         | ‚úÖ Correct                           |
| -------------------------------- | ------------------------------------ |
| I don't know what is the answer. | I don't know **what the answer is**. |
| She explained me the problem.    | She explained **the problem to me**. |

### 5. Preposition Errors

| ‚ùå Wrong       | ‚úÖ Correct        |
| -------------- | ----------------- |
| depend of      | depend **on**     |
| interested for | interested **in** |
| consist in     | consist **of**    |
| according with | according **to**  |

## üìä Grammar Practice Exercises

### Exercise 1: Transform to Passive

1. Scientists have discovered a new treatment. ‚Üí **\*\***\_\_\_**\*\***
2. The government will implement new policies. ‚Üí **\*\***\_\_\_**\*\***
3. People consider education essential. ‚Üí **\*\***\_\_\_**\*\***

<details>
<summary>Answers</summary>

1. A new treatment **has been discovered** (by scientists).
2. New policies **will be implemented** (by the government).
3. Education **is considered** essential.
 </details>

### Exercise 2: Combine Using Relative Clauses

1. The report was published yesterday. It shows positive trends.
2. Dr. Smith won the award. She works at Oxford.
3. The policy has failed. It was introduced last year.

<details>
<summary>Answers</summary>

1. The report **which/that was published yesterday** shows positive trends.
2. Dr. Smith, **who works at Oxford**, won the award.
3. The policy **that/which was introduced last year** has failed.
 </details>

### Exercise 3: Inversion Practice

Rewrite using inversion:

1. We have never faced such challenges before.
2. I realized the importance only when I lost it.
3. He not only failed but also gave up.

<details>
<summary>Answers</summary>

1. **Never before have we** faced such challenges.
2. **Only when I lost it did I** realize the importance.
3. **Not only did he** fail, **but he also** gave up.
 </details>

### Exercise 4: Conditionals

Complete with correct form:

1. If governments **\_\_\_** (invest) more, education **\_\_\_** (improve).
2. Had they **\_\_\_** (know), they **\_\_\_** (act) differently.
3. Were I **\_\_\_** (be) in charge, I **\_\_\_** (change) the policy.

<details>
<summary>Answers</summary>

1. If governments **invested** more, education **would improve**. (2nd conditional)
2. Had they **known**, they **would have acted** differently. (3rd conditional - inverted)
3. Were I **to be** in charge, I **would change** the policy. (2nd conditional - inverted)
 </details>

## üî§ Punctuation Rules for Band 8.5+

### Comma Usage

| Rule                           | Example                                                    |
| ------------------------------ | ---------------------------------------------------------- |
| **After introductory phrases** | **Consequently**, the project was delayed.                 |
| **Non-defining clauses**       | My sister**,** who lives in Canada**,** is a doctor.       |
| **Before coordinating conj.**  | I studied hard**,** but I still failed.                    |
| **In lists**                   | Education**,** healthcare**,** and housing are priorities. |

### Semicolon Usage

| Rule                        | Example                                                 |
| --------------------------- | ------------------------------------------------------- |
| **Between related clauses** | Some people prefer cities**;** others enjoy rural life. |
| **Before conjunctive adv.** | The plan was costly**;** however, it proved effective.  |

### Colon Usage

| Rule                          | Example                                                   |
| ----------------------------- | --------------------------------------------------------- |
| **Before a list/explanation** | There are three key factors**:** cost, time, and quality. |
| **Before elaboration**        | The solution is simple**:** invest more in education.     |
