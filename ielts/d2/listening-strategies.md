## ğŸ”‘ KEYWORD Recognition

### What are Keywords?

-   Content words that carry meaning
-   Usually nouns, verbs, adjectives
-   Help you predict and locate answers

### Before You Listen

1. **Read questions carefully** (use 30-second preview)
2. **Underline keywords**
3. **Think of synonyms** (they will paraphrase!)
4. **Predict answer type** (name? number? place?)

## ğŸ¯ Question Types

### 1. Form/Note Completion

**What:** Fill in gaps with words/numbers
**Tips:**

-   Word limit matters! (e.g., "NO MORE THAN TWO WORDS")
-   Spelling must be exact
-   Check grammar fits

**Example:**

> Name: John **\_**
> Contact: 0412 **\_**
> Booking date: 15th **\_**

### 2. Multiple Choice (MCQ)

**What:** Choose correct option(s)
**Tips:**

-   Read ALL options before audio
-   Listen for synonyms of options
-   Wrong answers (distractors) are often mentioned!

**Common Traps:**

-   Speaker mentions option A but then says "actually, B is better"
-   Numbers/dates from wrong options are stated
-   Speaker changes their mind

### 3. Matching

**What:** Match items from two lists
**Tips:**

-   Read both lists before listening
-   Order in audio may differ from questions
-   Answers may come quickly together

### 4. Map/Plan Labeling

**What:** Label locations on a diagram
**Tips:**

-   Start from where the speaker says (e.g., "entrance")
-   Follow directions: left, right, opposite, next to
-   Practice compass directions: north, south, east, west

### 5. Sentence Completion

**What:** Complete sentences with info from audio
**Tips:**

-   Grammar must fit
-   Look at words before/after gap
-   Answer is usually in same order as audio

## ğŸ“ Signal Words for Answers

### Listen for These Cues

| Signal                     | Meaning                     |
| -------------------------- | --------------------------- |
| "The main point is..."     | Key information coming      |
| "What's important is..."   | Answer likely follows       |
| "Actually..."              | Correction - new answer     |
| "I mean..."                | Clarification - real answer |
| "In fact..."               | Emphasis - key info         |
| "Don't forget..."          | Important detail            |
| "The thing is..."          | Key point follows           |
| "What I want to say is..." | Main message                |

### Contrast Signals

| Signal                 | Watch For           |
| ---------------------- | ------------------- |
| "But..."               | Change of direction |
| "However..."           | Opposite coming     |
| "Although..."          | Contradiction       |
| "On the other hand..." | Alternative view    |
