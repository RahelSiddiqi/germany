## üéØ SECTION-SPECIFIC STRATEGIES (What Actually Works)

### Listening (Target: 7.0+)

**The Reality:**

-   Listening is EASIEST to improve
-   Most students get 7.0+ with 2 weeks practice
-   Key: Exposure to different accents

**Proven Techniques:**

1. **Prediction Strategy**

    - Read questions BEFORE audio plays
    - Predict answer type (number? name? place?)
    - 80% of answers are predictable

2. **Keyword Listening**

    - Underline keywords in questions
    - Listen for synonyms (they NEVER use exact words)
    - Example: Question says "costly" ‚Üí Audio says "expensive"

3. **Focus on Numbers/Spelling**

    - Section 1 always has dates, phone numbers, addresses
    - Practice writing while listening
    - Don't panic during spelling sections

4. **Multiple Choice Trick**
    - Eliminate obviously wrong answers first
    - Audio mentions ALL options (trap!)
    - Correct answer = main point, not just mentioned

**Daily Practice:**

-   2 listening tests (60 minutes)
-   Review every mistake
-   Note down synonyms used
-   Goal: 30-35/40 correct = Band 7.5

**Resources:**

-   Cambridge IELTS 13-18 (official)
-   IELTS Liz website (free practice)
-   BBC Learning English (accent exposure)

---

### Reading (Target: 7.0+)

**The Reality:**

-   Reading is TIME PRESSURE game
-   You have 60 minutes for 3 passages (40 questions)
-   Most people can answer correctly if unlimited time - problem is SPEED

**Proven Time Management:**

```
Passage 1 (easiest): 17 minutes (13-14 questions)
Passage 2 (medium): 20 minutes (13-14 questions)
Passage 3 (hardest): 23 minutes (13-14 questions)
= 60 minutes total
```

**Strategies by Question Type:**

1. **True/False/Not Given (Hardest!)**

    - True = same meaning as text
    - False = opposite meaning
    - Not Given = no information (NOT "maybe true")
    - Trick: Don't use outside knowledge!

2. **Matching Headings**

    - Do FIRST (gives overview of passage)
    - Read first + last sentence of each paragraph
    - Match main idea, not details

3. **Sentence Completion**

    - Answers are IN ORDER in text
    - Use exact words from passage
    - Max 3 words usually

4. **Multiple Choice**
    - Eliminate wrong answers first
    - Correct answer = paraphrased from text
    - All options mentioned ‚â† all are correct

**Skimming Technique:**

1. Read title + any images (10 seconds)
2. Read first paragraph (main idea) (30 seconds)
3. Read first sentence of each paragraph (1 minute)
4. Now you know structure!

**Scanning Technique:**

1. Read question first
2. Find keywords
3. Scan text for those keywords (or synonyms)
4. Read that sentence + surrounding context
5. Answer question

**Daily Practice:**

-   3 passages timed (60 minutes)
-   Review mistakes
-   Learn new vocabulary (10-15 words/day)
-   Goal: 32-35/40 correct = Band 7.5

### Writing (Target: 7.0+)

**The Reality:**

-   Writing is HARDEST to improve alone
-   You NEED feedback from experienced teachers
-   But you CAN reach Band 7.0 with self-study

**Task 1 (20 minutes, 150+ words)**

**Types:**

1. Line graph (trends over time)
2. Bar chart (comparisons)
3. Pie chart (proportions)
4. Table (data comparison)
5. Process diagram (steps)
6. Map (changes over time)

**Winning Structure:**

```
Introduction (1 sentence): Paraphrase the question
Overview (2 sentences): Main trends/features (MOST IMPORTANT!)
Body 1 (3-4 sentences): First main feature + details
Body 2 (3-4 sentences): Second main feature + details

Total: 150-180 words (NOT more!)
```

**Band 7.0 Vocabulary:**

-   Increase: rise, grow, climb, surge, peak
-   Decrease: fall, drop, decline, plummet, dip
-   Stay same: remain stable, plateau, level off
-   Comparison: higher than, lower than, the same as, twice as much

**Common Mistakes:**

-   ‚ùå Giving opinions (DON'T! Just describe data)
-   ‚ùå Missing overview (AUTO Band 5!)
-   ‚ùå Including every number (select key features only)
-   ‚ùå Over-writing (150-180 is perfect, NOT 250!)

**Task 2 (40 minutes, 250+ words)**

**Essay Types:**

1. Opinion (Do you agree/disagree?)
2. Discussion (Discuss both views + your opinion)
3. Problem-Solution
4. Two-part question
5. Advantages vs Disadvantages

**Winning Structure:**

```
Introduction (2 sentences):
- Paraphrase question
- State your position clearly

Body 1 (5-6 sentences):
- Topic sentence (main idea 1)
- Explanation
- Example
- Result/Conclusion

Body 2 (5-6 sentences):
- Topic sentence (main idea 2)
- Explanation
- Example
- Result/Conclusion

Conclusion (2 sentences):
- Restate position
- Final thought

Total: 270-300 words
```

**Band 7.0 Requirements:**
‚úÖ Clear position throughout
‚úÖ 2-3 well-developed ideas (NOT 5 weak ideas)
‚úÖ Specific examples (NOT generic)
‚úÖ Cohesive devices (However, Moreover, Furthermore)
‚úÖ Topic-specific vocabulary
‚úÖ Mix of simple + complex sentences
‚úÖ Few grammar mistakes (NOT zero!)

**Daily Practice:**

-   Task 1: 1 essay (20 mins timed)
-   Task 2: 1 essay (40 mins timed)
-   Compare with model answers
-   Identify gaps
-   Goal: Clear structure, minimal errors

**Getting Feedback:**

-   IELTS Liz website (free correction service)
-   r/IELTS on Reddit (free peer review)
-   Fiverr (¬£5-10 per essay from IELTS teachers)
-   Worth it for 5-10 essays to understand mistakes!

### Speaking (Target: 7.0+)

**The Reality:**

-   Speaking is most UNPREDICTABLE
-   Depends on examiner, topic, your confidence
-   But Band 7.0 is VERY achievable with practice

**Test Structure:**

**Part 1 (4-5 minutes): Introduction**

-   Questions about: work, study, hometown, hobbies, daily routine
-   Keep answers 2-3 sentences
-   Be natural, friendly

**Example:**

```
Q: Do you work or study?
‚ùå Bad: "I study computer science."
‚úÖ Good: "I'm currently studying computer science at XYZ University. I'm in my final year and really enjoying my courses on cybersecurity and networking."
```

**Part 2 (3-4 minutes): Long turn**

-   You get a cue card with a topic
-   1 minute to prepare notes
-   Speak for 2 minutes
-   Examiner asks 1-2 follow-up questions

**Cue Card Strategy:**

```
Who/What/Where: 20 seconds
When/Why: 20 seconds
How: 30 seconds
Your feelings/thoughts: 30 seconds
Conclusion: 20 seconds
= 2 minutes!
```

**Golden Topics (prepare these!):**

1. Describe a person you admire
2. Describe a place you visited
3. Describe an important decision
4. Describe a skill you learned
5. Describe a book/movie
6. Describe a childhood memory
7. Describe your future goal
8. Describe a technological device
9. Describe an achievement
10. Describe a problem you solved

**Part 3 (4-5 minutes): Discussion**

-   Abstract questions related to Part 2 topic
-   Longer, more complex answers needed
-   Show critical thinking

**Example:**

```
Q: How has technology changed education?
‚ùå Bad: "Technology is good for education."
‚úÖ Good: "Technology has revolutionized education in several ways. Firstly, online learning platforms like Coursera have made quality education accessible globally. However, there are challenges too, such as the digital divide where students in rural areas might lack internet access. Overall, I believe the benefits outweigh the drawbacks, but we need to ensure equitable access."
```

**Band 7.0 Requirements:**
‚úÖ Speak at length without noticeable effort
‚úÖ Use discourse markers (Firstly, However, On the other hand)
‚úÖ Paraphrase effectively
‚úÖ Use less common vocabulary (occasionally is fine!)
‚úÖ Mix simple + complex grammar
‚úÖ Clear pronunciation (accent is OK!)
‚úÖ Some hesitation/errors are OK (it's about communication!)

**Daily Practice:**

-   Part 1: Answer 10 questions (15 mins)
-   Part 2: Practice 2 cue cards (2 mins each, record yourself)
-   Part 3: Discuss 3 abstract topics (5 mins each)
-   Listen to recordings, improve fluency
-   Goal: Speak confidently for 2 minutes

**Practice Partners:**

-   iTalki (¬£5-10/hour native speakers)
-   Language exchange apps (free: Tandem, HelloTalk)
-   IELTS speaking practice groups (Facebook, Reddit)
-   Record yourself and review

## üéØ COMMON MISTAKES TO AVOID

### Listening

‚ùå Writing answers while listening (miss next question)
‚úÖ Focus on listening, write during pauses

‚ùå Panicking if you miss one answer (spiral effect)
‚úÖ Move on immediately, focus on next question

‚ùå Changing answers at the end (usually first instinct correct)
‚úÖ Only change if you're 100% certain

### Reading

‚ùå Reading entire passage first (waste time)
‚úÖ Skim quickly, then answer questions

‚ùå Using outside knowledge for True/False/Not Given
‚úÖ Only use information in the passage

‚ùå Spending too long on one question (miss others)
‚úÖ If stuck after 1 minute, guess and move on

### Writing

‚ùå Memorizing templates word-for-word (examiner spots this!)
‚úÖ Learn flexible structures, adapt to each question

‚ùå Using very complex words incorrectly (looks bad)
‚úÖ Use vocabulary you're comfortable with correctly

‚ùå Not planning (messy structure)
‚úÖ Spend 2-3 minutes planning before writing

### Speaking

‚ùå Memorizing full answers (sounds robotic)
‚úÖ Prepare ideas/vocabulary, speak naturally

‚ùå Long pauses/silence (hurts fluency)
‚úÖ Use fillers: "Well...", "That's a good question...", "Let me think..."

‚ùå Giving very short answers (Part 1)
‚úÖ Extend answers: answer + reason/example
